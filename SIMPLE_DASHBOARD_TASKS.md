## 🎨 레이아웃 설계
```
┌─────────────────────────────────────┐
│ 📁 프로젝트 관리                    │ ← ProjectSection
│ [프로젝트 선택 ▼] [+ 새 프로젝트]   │
├─────────────────────────────────────┤
│ 📋 테이블 관리                      │ ← TableSection  
│ □ 테이블1 □ 테이블2 [+ 새 테이블]   │
├─────────────────────────────────────┤
│ ⚙️ 컬럼 편집 (ERD 스타일)           │ ← ColumnEditor
│ ┌─────┬──────┬─────┬─────┬─────┐    │
│ │컬럼명│타입  │길이 │NULL │설명 │    │
│ ├─────┼──────┼─────┼─────┼─────┤    │
│ │ id  │BIGINT│     │  □  │기본키│    │
│ └─────┴──────┴─────┴─────┴─────┘    │
├─────────────────────────────────────┤
│ ✅ 검증 결과                        │ ← ValidationSection
│ [실시간 검증 피드백]                │
├─────────────────────────────────────┤
│ 📤 스키마 내보내기                  │ ← ExportSection
│ [형식 선택] [다운로드]              │
└─────────────────────────────────────┘
```

## 🔧 기술 스택
- **Frontend**: React 19 + TypeScript + Vite + Tailwind CSS
- **상태 관리**: Zustand (기존 stores 재사용)
- **API**: 기존 백엔드 API 그대로 활용
- **라우팅**: React Router (새 경로 `/simple` 추가)
