# 제품 개요

## MSSQL 데이터베이스 모델링 도구

MSSQL 데이터베이스 스키마의 직관적인 설계와 관리를 가능하게 하는 웹 기반 데이터베이스 모델링 플랫폼입니다. 테이블 생성부터 스키마 내보내기까지 완전한 데이터베이스 모델링 라이프사이클을 지원합니다.

### 핵심 기능
- **간소화 버전 (SimpleDashboard)**: 원페이지 통합 대시보드로 모든 기능 접근
- **ERD 스타일 편집**: 20-column 그리드 인라인 편집으로 빠른 컬럼 관리
- **MSSQL 전문 지원**: IDENTITY, DECIMAL 정밀도/스케일 등 27개 데이터 타입
- **한글명 관리**: 영문명과 한글명 이중 관리 시스템
- **실시간 검증**: 명명 규칙 위반 즉시 감지 및 수정 제안
- **인덱스 관리**: 클러스터드/논클러스터드, 복합 인덱스 지원
- **스키마 내보내기**: SQL, JSON, Markdown, HTML, CSV 다중 형식 지원
- **PostgreSQL 백엔드**: Clean Architecture 기반 안정적인 데이터 영속성

### 대상 사용자
- 데이터베이스 설계자 및 아키텍트
- 일관된 데이터베이스 표준이 필요한 개발팀
- 데이터베이스 설계 프로세스를 감독하는 프로젝트 매니저

### 주요 가치 제안
- **빠른 작업 속도**: ERD 스타일 인라인 편집으로 exerd 수준의 편의성
- **한글 친화적**: 한글명과 영문명을 별도 관리하여 국내 환경에 최적화
- **실시간 검증**: 명명 규칙 위반을 즉시 감지하여 오류 사전 방지
- **자동화된 SQL**: MSSQL 전용 DDL 스크립트 자동 생성으로 수동 오류 제거
- **원페이지 UI**: 모든 기능이 한 화면에 통합되어 빠른 접근 가능
- **영속적 저장**: PostgreSQL 기반 안정적인 프로젝트 관리
